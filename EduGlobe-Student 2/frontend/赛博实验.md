## 🎯 模块目标

“赛博实验”是一款聚焦中学地理核心知识点的虚拟实验工具，通过 **高仿真可视化交互场景** 将抽象的地理原理转化为直观可操作的实验过程。  
系统围绕“实验选择 → 变量调控 → 过程观察 → 结果分析” 四步科学探究流程构建。

## 🧱 核心功能模块设计

### 一、实验库模块（index.vue）

**功能**
- 展示系统收录的基础实验与拓展实验
- 支持“知识模块筛选”与“关键词搜索”
- 每个实验卡片展示：
  - 实验标题
  - 对应知识模块（如“大气运动”）
  - 教材关联章节（如《地表形态的塑造》）
  - 标签（知识点 / 适用年级）

**实验示例**
1. 热力环流模拟实验  
2. 大气保温作用模拟实验  
3. 水土流失模拟实验  
（可扩展：地表径流影响因素实验）

点击实验卡片 → 跳转 `/feature/cyberlab/experiment` 并传递实验 ID。

---

### 二、变量调控模块（VariableControlPanel.vue）

**功能**
- 每个实验提供 3–4 个核心变量。
- 变量通过按钮组或滑块呈现，每个选项带图文说明。
- 变量切换时，实时触发场景更新。

**通用规则**
- 仅控制单一变量（其余保持默认值）
- 变量变化 → 动态更新实验场景（颜色、密度、角度、速率等）

**示例变量设计**

#### 热力环流实验
- 热源强度（30℃ → 60℃ → 100℃）
- 冷源强度（-20℃ → 0℃ → 20℃）
- 空间密闭性（开放 / 密闭）

#### 大气保温作用实验
- 大气浓度（低 / 中 / 高）
- 光照强度（弱 / 中 / 强）
- 昼夜状态（白天 / 黑夜）

#### 水土流失实验
- 植被覆盖（无 / 稀疏 / 茂密）
- 地面坡度（0° / 30° / 60°）
- 降水强度（小雨 / 中雨 / 大雨 / 暴雨）
- 土壤性质（砂质 / 壤质 / 黏质）

### 三、实验交互模块（ExperimentScene.vue）

**功能**
- 使用 Three.js 创建动态可视化实验场景
- 实现以下交互：
  - ▶️ 开始实验（变量触发动画运行）
  - ⏸ 暂停实验（冻结当前帧）
  - 🔁 重置实验（恢复初始状态）
  - 📘 实验原理（弹出 PrinciplePopup）

**画面逻辑示例**

#### 1️⃣ 热力环流模拟
- 场景：矩形透明空间，左红右蓝，顶部白色流线表示气流
- 动态效果：
  - 热源升温 → 红区上升气流更快，流线密集
  - 冷源增强 → 蓝区下沉气流加快
  - 形成闭合循环（热升冷降）

#### 2️⃣ 大气保温作用
- 场景：两个透明容器（无大气组 / 有大气组）
- 动态变化：
  - 光照开启时升温，两组温差逐渐拉开
  - 夜间关闭光照后，有大气组降温缓慢
  - 温度实时显示在温度计上（每秒更新）

#### 3️⃣ 水土流失模拟
- 场景：对照组与实验组并列土壤剖面
- 动态效果：
  - 雨滴密度、坡度、植被、颜色实时变化
  - 模拟降水与径流动画（粒子效果）
  - 底部槽液浑浊度随流失量变化加深

### 四、结果分析模块（ResultChart.vue）

**功能**
- 实验结束后自动生成对比数据与可视化图表
- 包括：
  1. 实时数值展示（对照组 vs 实验组）
  2. 柱状对比图（ECharts）
  3. 趋势曲线（如温度变化随时间）
  4. 差值标注 + 箭头趋势
  5. 数据解读文字说明

**样例计算逻辑**
```js
// 热力环流
环流范围差异 = (实验组半径 - 对照组半径) / 对照组半径 * 100

// 水土流失
减少百分比 = (实验组流失量 - 对照组流失量) / 对照组流失量 * 100

// 大气保温
保温提升率 = (对照组降温幅度 - 实验组降温幅度) / 对照组降温幅度 * 100
```

**异常数据检测**
- 若出现负值或异常波动 → 弹出提示框：
  ```
  ⚠️ 数据异常，请检查变量设置。
  ```

### 五、实验原理模块（PrinciplePopup.vue）

**功能**
- 弹出式知识点解析窗口
- 内容严格依据中学地理教材与权威资料
- 原理说明精准且结合实验场景（带图标与标注）

**示例内容**
- 水土流失原理：
  “植被通过根系固定土壤、叶片减缓雨滴冲击力，可显著降低流失量；
  坡度越大、降水越强，水流冲击力越大，侵蚀越严重；
  黏质土壤抗侵蚀能力最强。”
