## 🎯 模块目标

“情景互动”模块是一款以“游戏化交互 + 地理场景深度融合”为核心的 **第一视角沉浸式地理探索系统**。  
通过虚拟角色化体验、动态地理场景、任务驱动学习和多维度交互，帮助学生在“行走式探索”中掌握地理知识，实现从被动听讲到主动探究的转变。

## 🧱 核心功能模块设计

### 一、核心体验：第一视角角色化沉浸探索

#### 1️⃣ 角色形象定制系统（AvatarCustomizer.vue）
- 支持性别、发型、面部特征自定义；
- 提供“职业套装”库（地质学家、气象观测员等）；
- 装备随场景变化自动调整：
  - 热带雨林 → 防刺靴、驱蚊面罩；
  - 极地冰川 → 防风科考服、冰镐、冰爪。

#### 2️⃣ 第一视角控制体系（FirstPersonView.vue）
- 操作布局：左侧虚拟摇杆 + 右侧功能按键；
- 支持动作：行走、奔跑、攀爬、涉水；
- 动态反馈：
  - 坡度影响移动速度；
  - 水流场景带阻力反馈。

右侧四大操作键：
- **探索键**：触发聚焦观测 + 知识点弹窗 + 录音记录；
- **工具键**：调用虚拟地理工具（海拔仪、采样器、扫描仪）；
- **记录键**：拍照、标注、绘制地图、文字笔记；
- **任务键**：查看当前任务清单与奖励进度。

---

### 二、核心功能：场景化地理知识探索

#### 1️⃣ 动态地理场景库（SceneMap.vue）
- 场景分类：自然地理 / 人文地理；
- 覆盖核心知识点（火山喷发、季风环流、喀斯特溶洞、城市功能区、聚落选址等）；
- 每个场景包含：
  - 场景描述与进入条件；
  - 动态地理要素（气候、地形、生态等）；
  - 实时交互（触摸、观测、测量）。

**场景示例**
- 火山喷发观测：震动反馈 + 温度变化 + 岩浆流动动画；
- 季风环流体验：时间轴切换冬夏气流 + 风速与降雨动态；
- 喀斯特溶洞探险：手电筒照明 + 特征识别 + 地下暗河探索；
- 城市功能区模拟：热力图、风向动态、污染扩散模型；
- 聚落选址挑战：不同地形风险提示 + 虚拟降雨验证。

---

#### 2️⃣ 知识触发与验证系统（KnowledgePopup.vue）
- 每个场景内设置 20–30 个隐藏交互点；
- 触摸或靠近触发知识点弹窗：
  - 说明 + 插图 + 原理动画；
- 支持“数据验证”功能：操作验证地理规律；
  - 例：洋流影响气候 → 比较暖流/寒流区域温度与降水。

---

#### 3️⃣ 探索任务体系（TaskPanel.vue）
任务分级：
- **基础观测任务**：地形识别与描述；
- **综合分析任务**：原理应用与问题解决；
- **创新探究任务**：自设变量 + 实验验证 + 数据导出。

**任务示例**
- 等高线任务：识别山顶/山谷/鞍部/陡崖；  
- 台风应对任务：避难点选择 + 应急物资准备评分；  
- 植被实验任务：自设覆盖率模拟水土流失差异。

任务完成后生成报告（JSON/Markdown），并触发奖励系统（徽章、称号、工具解锁）。

---

### 三、核心特色

#### 1️⃣ 地理真实性 + 游戏趣味性
- 场景数据基于真实地理资料（中科院地理资源所、NASA DEM等）；
- 动态气候、海拔、洋流数据真实可视化；
- 奖励机制：探索等级、成就徽章、排行榜系统；
  - 示例徽章：“地理探索者”“地貌识别大师”“地理之星”。

#### 2️⃣ 多设备适配
- 移动端：摇杆控制 + 多点触控；
- VR端：第一视角沉浸体验（360°观察）；
- PC端：键盘WASD + 鼠标控制。

#### 3️⃣ 跨场景数据联动
- 河流场景采集的泥沙数据可导入土壤场景；
- 城市热岛结果可关联气候类型场景。

#### 4️⃣ 校本化扩展
- 教师可上传本地数据，生成“家乡地理场景”；
- 支持自定义任务与地形；
- 自动匹配学生兴趣与薄弱点，推送个性化探索路径。

## 🎨 技术与交互规范

- 前端框架：Vue3 + Vite + TypeScript + Tailwind  
- 3D引擎：Three.js（支持移动端渲染优化）  
- 动画库：Framer Motion  
- 图表展示：ECharts  
- 场景交互：Raycaster 点击检测 + 物理反馈  
- 操作控制：虚拟摇杆（移动端） / WASD（PC）  
- 数据持久化：localStorage（探索进度与日志）  
- 导出格式：Markdown / JSON 报告  
- 移动端适配：flex 布局 + vw 单位
