## ⚡ 快速检查

```powershell
# 检查所有服务是否运行
netstat -ano | findstr "3000 3306 5173"

# 查看 Node 进程
Get-Process node

# 检查 MySQL
Get-Process | Where-Object { $_.ProcessName -eq "mysqld" }
```

## ❓ 常见问题

### MySQL 启动失败
```powershell
# 停止冲突的 MySQL
Get-Process | Where-Object { $_.Path -like "*mysql*" } | Stop-Process -Force
# 然后重新启动
.\start-mysql.ps1
```

### 端口被占用
```powershell
# 查看端口占用
netstat -ano | findstr :3000
# 停止进程（替换 <PID>）
Stop-Process -Id <PID> -Force
```

### 后端无法连接数据库
1. 确认 MySQL 正在运行
2. 检查密码是否正确
3. 验证 .env 配置
