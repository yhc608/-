## 🔧 技术实现

### 数据库设计
- **总数据表**: 60+ 张表
- **核心表**: 用户、学校、知识图谱等基础表
- **功能表**: 四大模块独立表结构
- **关联表**: 标签、评论、收藏等辅助表
- **分析表**: 学习轨迹、数据分析等统计表

### 后端架构
```
backend/server/src/
├── index.ts                    # 主入口
├── config/
│   └── database.ts             # 数据库连接池
├── middleware/
│   ├── auth.middleware.ts      # JWT认证
│   ├── error.middleware.ts     # 错误处理
│   ├── logger.middleware.ts    # 日志记录
│   └── rate-limit.middleware.ts # 限流
├── routes/
│   ├── auth.routes.ts          # 认证路由
│   ├── user.routes.ts          # 用户路由
│   ├── geopolitics.routes.ts   # 经纬瞰政
│   ├── cyberlab.routes.ts      # 赛博实验
│   ├── scenario.routes.ts      # 情景互动
│   └── ai-companion.routes.ts  # AI伴学
├── controllers/
│   └── *.controller.ts         # 控制器
└── services/
    └── *.service.ts            # 业务逻辑
```

### 前端架构
```
frontend/src/
├── main.ts                     # 入口文件
├── App.vue                     # 根组件
├── router/
│   └── index.ts                # 路由配置
├── services/
│   └── api.ts                  # API调用（已完整实现）
├── pages/
│   └── feature/
│       ├── geopolitics/        # 经纬瞰政页面
│       ├── cyberlab/           # 赛博实验页面
│       ├── interactive/        # 情景互动页面
│       └── companion/          # AI伴学页面
└── components/
    └── common/                 # 通用组件
```

### API设计规范
- **RESTful风格**: 符合REST标准
- **统一响应格式**: `{ success, data, message }`
- **JWT认证**: 基于Token的身份验证
- **错误处理**: 统一异常捕获和返回
- **日志记录**: 完整的请求日志

```powershell

npm run dev

# 3. 启动前端

npm install
npm run dev
```
