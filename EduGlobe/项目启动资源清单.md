## 📋 启动顺序说明

### 自动启动（推荐）

使用 `快速启动.bat` 或 `start-all.ps1`：
- 会自动按正确顺序启动所有服务
- 会检查服务是否已在运行（避免重复启动）
- 会显示详细的启动进度

### 手动启动

如需手动控制，按以下顺序：

```powershell
# 1. 启动 MySQL（数据层）
.\start-mysql.ps1

# 2. 启动后端（业务逻辑层）
cd backend/server
npm start

# 3. 启动前端（展示层）
cd frontend
npm run dev
```

## 🔧 自定义配置

### 修改 MySQL 路径

如果你的 MySQL 不在 `D:\mysql-8.0.23-winx64`，需要修改以下文件：

1. `start-mysql.ps1` - 第 12 行
2. `stop-mysql.ps1` - 第 8 行  
3. `start-all.ps1` - 第 13 行
4. `stop-all.ps1` - 第 9 行
5. `check-environment.ps1` - 第 98 行

修改这一行：
```powershell
$MYSQL_PATH = "D:\mysql-8.0.23-winx64\bin\mysqld.exe"
```

### 修改数据库密码

编辑 `backend/server/.env` 文件：
```env
DB_PASSWORD=你的新密码
```
